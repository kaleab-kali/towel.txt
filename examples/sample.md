# Towel.txt Sample Document

This sample shows the printable HTML output generated from Markdown.

## Overview

Towel.txt keeps documents readable in source form while producing a cleaner
browser-printable version for sharing.

## Checklist

- Write the source document in Markdown.
- Generate HTML with the CLI.
- Open the HTML file in a browser.
- Print or save to PDF from the browser.

## Quote

> Simple documents should stay simple all the way to print.

## Table

| Section           | Purpose                      |
| ----------------- | ---------------------------- |
| Heading anchors   | Linkable document sections   |
| Table of contents | Fast navigation              |
| Print CSS         | Cleaner paper and PDF output |

## Code

```bash
towel-txt examples/sample.md -o examples/sample.html
```
